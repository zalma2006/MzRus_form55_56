Эта программа создана для обработки данных из 55 и 56 форм МЗ РФ, приведения каждой формы отчётности к определённому шаблону. Мы располагаем данными с 2015 года по 2020 год, за каждый регион, в общей сложности 1020 файлов, формата word, которые мы преобразуем в формат .xlsx. Это 
В 55 форме отчётности в общей сложности представлены 11 таблиц:
1. Сведения о центре медицины катастроф (далее МК)
2. Сведения о кадрах центра МК
3. Формирования службы МК
4. Сведения о ЧС по числу пострадавших и характере тяжести
5. Сведения о ЧС по видам оказанной помощи
6. Использование коечного фонда медицинских организаций в чрезвычайных ситуациях (ЧС)
7. Сведения о работе лаборатории психофизиологического обеспечения центра медицины катастроф
8. Сведения о проведении обучения
9. Сведения о проведенных учениях, тренировках и занятиях
10. Сведения о деятельности трассовых пунктов экстренной медицинской помощи
11. Сведения о материально-технической оснащенности центра медицины катастроф

После этого будет проводиться агрегация сводных данных, разделение на группы и их сравнение, для использования в
целях публикации научной статьи.
